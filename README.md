Gra	- Gun Flames
======
Prosta strzelanka połączona z elementami zręcznościówki. Zabijanie wrogów, niszczenie ścian, unikanie pułapek.
------

### Instrukcja

### Poruszanie się
Lewo - Przycisk `A` lub `<-`
Prawo - Przycisk `D` lub `->`
Skok - Przycisk `W` lub `Spacja`
Strzelanie - `LPM` - lewy przycisk mysz
